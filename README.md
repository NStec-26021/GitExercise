# Numer0n

## 実行方法

### IDE

IDEから実行する場合は、`app/src/main/java/numer0n/App.java`を実行してください。

## ゲームルール

- ランダムに生成する3桁の数字を当てる。
- 各桁は0-9の数字で、重複しない。
- 数字と桁が合っていればEAT、数字が合っていればBITEとする。
- EATとBITEの数を提示し、次の数字を当てる。
- 10回以内に3桁を当てれば勝利とする。

## ゲームフロー

1. ゲームを開始するか問い合わせる。
2. 開始する場合は、ランダムに3桁の数字を生成する。
3. プレイヤーに3桁の数字を入力させる。
4. 入力した数字と生成した数字を比較し、EATとBITEの数を提示する。
5. 10回以内に3桁を当てれば勝利とする。
6. 10回以内に3桁を当てられなかった場合は敗北とする。
7. ゲームを続けるか問い合わせる。
8. 続ける場合は、2へ。
9. 続けない場合は、ゲームを終了する。

## アイテム（未実装）

### 共通仕様

- 初期状態でアイテムはすべて所持
- 各アイテムは1回のみ使用可能
- アイテムを使用できる場合は、アイテムの使用結果を表示し、ターンを1回消費する。
- アイテムを使用できない場合は、使用できない旨を表示する。

### High & Low

- 各桁がそれぞれHIGH(5-9)かLOW(0-4)かを判定する。

### Slash

- 各桁の中の最大値から最小値を引いた値を表示する。（最大値7、最小値2の場合、5）
